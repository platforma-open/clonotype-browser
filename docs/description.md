# Overview

Provides an interactive interface for browsing, exploring, and annotating sequences across samples. The block accepts T/B cell receptor clonotypes from V(D)J sequencing or peptides from display-selection campaigns, and enables users to examine sequence characteristics, frequencies, and distributions.

The block supports custom annotation scripts that allow users to tag sequences based on various criteria such as frequency thresholds, sequence characteristics, or sample-specific patterns. Annotated sequences can be tracked, filtered, and exported for downstream analysis. The interface provides multiple views including per-sample sequence tables, overlap analysis between samples, and statistical summaries of sequence distributions.
